# Internship Tracker Pro

A full-stack internship management platform built with Flask and MongoDB Atlas.

## Features

* User Registration & Login
* Secure Password Hashing
* Session Authentication
* Add Applications
* Edit Applications
* Delete Applications
* Search Applications
* Filter by Status
* Analytics Dashboard
* Chart.js Visualizations
* User-specific Data

## Tech Stack

* Python
* Flask
* MongoDB Atlas
* HTML
* CSS
* JavaScript
* Chart.js

## Installation

```bash
git clone <repository-url>
cd internship-tracker-pro
pip install -r requirements.txt
python app.py
```

## Future Improvements

* CSV Export
* Dark Mode
* Email Notifications
* AI Resume Analysis
